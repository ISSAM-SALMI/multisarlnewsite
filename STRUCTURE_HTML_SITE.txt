═══════════════════════════════════════════════════════════════════════════════
                         2. NAVBAR (BARRE DE NAVIGATION)
═══════════════════════════════════════════════════════════════════════════════

STRUCTURE:

<header> (Fixed en haut, largeur 100%)
  │
  └─ <div class="container"> (Conteneur centré avec padding)
      │
      └─ <div> (Flex, justify-between, align-center)
          │
          ├─ LOGO (Lien + SVG)
          │   └─ <a href="#">
          │       ├─ <div> (Effet de glow au hover)
          │       └─ <Logo /> (Composant SVG)
          │
          ├─ NAVIGATION DESKTOP (Caché sur mobile, visible lg:)
          │   └─ <nav> (Flex avec gap)
          │       └─ 5 liens:
          │           ├─ <a href="#accueil">Accueil</a>
          │           ├─ <a href="#services">Expertises</a>
          │           ├─ <a href="#vente-location">Location & Vente</a>
          │           ├─ <a href="#engagements">Engagements</a>
          │           └─ <a href="#contact">Contact</a>
          │
          ├─ CTA DESKTOP (Caché sur mobile, visible lg:)
          │   └─ <div> (Flex, gap-6)
          │       ├─ <div> (Infos contact)
          │       │   └─ <a href="tel:0522603621">
          │       │       └─ Icône Phone + "05 22 60 36 21"
          │       │
          │       └─ <a href="#contact"> (Bouton CTA)
          │           └─ "DEVIS GRATUIT" + Icône ChevronRight
          │
          └─ BOUTON MENU MOBILE (Visible mobile, caché lg:)
              └─ <button>
                  └─ Icône Menu ou X (toggle)

MENU MOBILE (Overlay plein écran quand ouvert):

<div> (Position fixed, hauteur 100vh, fond navy-900)
  │
  └─ <nav> (Flex column, centré)
      │
      ├─ 5 liens de navigation (mêmes que desktop)
      │
      ├─ <div> (Infos contact)
      │   └─ Téléphone + Email
      │
      └─ <a> (Bouton CTA)
          └─ "DEMANDER UN DEVIS"

NOMBRE D'ÉLÉMENTS NAVBAR:
- 1 logo
- 5 liens de navigation
- 1 numéro de téléphone
- 1 bouton CTA
- 1 bouton toggle menu mobile

                         3. SECTION HERO (#accueil)
═══════════════════════════════════════════════════════════════════════════════

STRUCTURE:

<div> (Position relative, hauteur 100vh minimum, fond navy-950)
  │
  ├─ IMAGE DE FOND (Position absolute, couvre toute la section)
  │   └─ <div style="background-image: url(...)">
  │
  ├─ OVERLAY GRADIENT (Position absolute, dégradé navy)
  │   └─ <div class="absolute inset-0">
  │
  ├─ PATTERN DE GRILLE (Position absolute, points décoratifs)
  │   └─ <div class="absolute inset-0">
  │
  ├─ CONTENU PRINCIPAL (Position relative, z-index supérieur)
  │   └─ <div class="container">
  │       └─ <div class="max-w-5xl">
  │           │
  │           ├─ BADGE DE STATUT
  │           │   └─ <div> (Inline-flex, badge arrondi)
  │           │       ├─ <span> (Point rouge clignotant)
  │           │       └─ "Intervention 24/7 au Maroc"
  │           │
  │           ├─ TITRE PRINCIPAL
  │           │   └─ <h1>
  │           │       ├─ "VOTRE PARTENAIRE"
  │           │       ├─ "ÉLECTRICITÉ" (avec dégradé)
  │           │       └─ "& TECHNOLOGIES CONNECTÉES" (texte outline)
  │           │
  │           ├─ DESCRIPTION
  │           │   └─ <p>
  │           │       ├─ Texte principal
  │           │       └─ <span> (Texte secondaire)
  │           │
  │           └─ BOUTONS D'ACTION
  │               └─ <div> (Flex column/row, gap-5)
